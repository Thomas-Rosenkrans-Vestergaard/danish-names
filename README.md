# Danish names

Contains statistics about the use of danish names taken from this [page](http://www.danskernesnavne.navneforskning.ku.dk/Topnavne.asp) created by the [Department of Nordic Studies and Linguistics](https://nors.ku.dk/english/) at the [University of Copenhagen](https://www.ku.dk/english/). The data is presented as both `json` or `csv`. The data is from 2005.

## Structure

The data is seperated into files based on the type of name (first, middle, last) and gender (all, male, female). The contents of each file is a list of names together with information about the frequency of the name by location.
